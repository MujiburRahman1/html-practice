HTML Entities

  • An html entity is a piece of text ("stirng") that begins with an ampersand (&) and ends with a semicolon (;).
  • Used to display resereved characters (which would otherwise be interpreted as HTML code), and invisibel characters (like non-breaking spaces).
  • Can also use in place of characters that are difficult to type with a standard keyboard.
  • Browser interprets them & renders correct character
  • Example: < will be displayed as < in the browser
  • HTML entities are used to represent characters that have special meaning in HTML, such as <, >, &, ", and '.
  • They are also used to represent characters that may not be easily typed on a keyboard, such as non-breaking spaces or special symbols.
  • HTML entities are written as an ampersand (&) followed by the entity name or number, and ending with a semicolon (;).
  • For example, the HTML entity for the less than symbol (<) is <, and for the greater than symbol (>) is >.
  • Using HTML entities helps ensure that the content is displayed correctly in web browsers and prevents any confusion with HTML tags.

HTML Entities Examples

  • Less than: <
  • Greater than: >
  • Ampersand: &
  • Double quote: "
  • Single quote: '
  • Copyright: ©
  • Registered: ®
  • Euro: €
  • Non-breaking space: Hello World
  • Trademark: ™
  • Cent: ¢
  • Pound: £
  • Yen: ¥
  • Section: §
  • Degree: °

Semantic Markup

Semantic markup is the use of HTML markup to reinforce the semantics, or meaning,
of the content on a web page, rather than to define its presentation or look.

Benefits of Semantic Markup

  • Improves accessibility for screen readers and other assistive technologies.
  • Enhances SEO by providing search engines with better context about the content.
  • Makes the code more readable and maintainable for developers.

Examples of Semantic Elements

  • <header> - Represents introductory content or navigational links.
  • <nav> - Defines a set of navigation links.
  • <article> - Represents a self-contained piece of content.
  • <section> - Defines a section in a document.
  • <footer> - Represents the footer of a document or section.

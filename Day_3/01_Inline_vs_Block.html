Block Elements

Block elements start on a new line and take up the full width available

This is a <div> (block element)
<div> is a block element

This is a <p> (block element)
<p> is a block element

This is a <h1> (block element)
<h1> is a block element

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary

This is a <span> (inline element)
<span> is an inline element This is an <a> (inline element)
<a> is an inline element This is a <strong> (inline element)
<strong> is an inline element
